Detalhes sobre tags html

Uma explicação breve sobre tags e como usa-las


Tags

h1 p mark small i u strong ol ul li a hr sub sup blockquote abbr

H1 - H6

As tags de H1 a H6 são tags utilizadas para títulos e para criar um destaque em relação a outras tags.


P

A tag P é utilizada para iniciar um parágrafo, normalmente utilizado para descrições após utilizar os títulos.


mark

A tag mark é feita para dar um destaque ao texto, criando um contraste, ela funciona dessa forma: Teste.


small

A tag small serve para diminuir um texto e funciona da seguinte forma: eu sou pequeno.


i

A tag i serve para deixar o texto que deseja em itálico visualmente, dessa forma: Esse é um texto em itálico.


u

A tag u serve para deixar o texto desejado sublinhado, dessa forma: Esse é um texto sublinhado.


strong

A tag strong serve para deixar um texto em negrito visualmente e também semânticamente, funciona dessa forma: Esse é um texto com destaque.


ol

A tag ol serve para criar uma lista ordenada, pode ser usada para listar passos de uma receita ou itens de compras, funciona da seguinte forma:

  1. Lapís e caneta
  2. Caderno
  3. Bolsa


ul

A tag ul serve para criar uma lista não ordenada, pode ser usada para citar pontos de uma argumentação, ou até mesmo para criar um menu de um site, funciona da seguinte forma:

  1. Não conseguem encaixar toda a história num período curto de tempo
  2. Não passam a mesma intensidade das histórias dos livros
  3. Na maioria das vezes alteram a história para se adequar ao universo cinematográfico


li

A tag li é usada para criar itens dentro de listas ordenadas e listas não ordenadas, elas são o que está destacado em amarelo:

  • Exemplo 1
  • Exemplo 2
  • Exemplo 3


a

A tag a é conhecida como anchor e pode te encaminhar para outras páginas, tanto de fora do site como do próprio site, além de facilitar a navegação em páginas, por exemplo: Clique aqui para voltar ao início dessa página


hr

A tag hr põe uma linha nesse estilo:


sub

A tag coloca um texto abaixo da linha de escrita, funciona nesse estilo: H2O


sup

A tag sup faz o contrário da sub, coloca o texto acima da linha de escrita, dessa forma: x2 = 4


blockquote

A tag blockquote é usada para citar frases com referências, exemplo:

"Não sou nada. Nunca serei nada. Não posso querer ser nada. À parte isso, tenho em mim todos os sonhos do mundo..."

abbr

A tag abbr é utilizada para criar um tooltip ao passar o mouse por textos selecionados, exemplo: O MEC Decidiu o dia de reaplicação de provas do ENEM 2022...
